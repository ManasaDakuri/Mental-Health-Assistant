# Mental Health Assistant

The Mental Health Assistant is a digital tool designed to support individuals in managing their mental health. It offers intelligent conversational assistance, early detection of mental health issues, and resources for emotional support using advanced Natural Language Processing (NLP) and machine learning.

## 🧠 Overview

This assistant simulates empathetic conversations to understand the user's emotional state and provide supportive responses. The goal is to improve accessibility to mental health guidance, reduce stigma, and offer timely suggestions for well-being.

## ✨ Features

- 🧘 Mood Detection using sentiment analysis
- 💬 Chatbot for mental health support
- 📚 Resource suggestions based on user needs
- 🚥 Severity Level Assessment (Mild, Moderate, Severe)
- 📈 Data analytics using word clouds and sentiment charts

🛡️ Disclaimer
This tool is not a replacement for professional therapy or diagnosis. It is designed for educational and supportive purposes only.
